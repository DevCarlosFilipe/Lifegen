# Lifegen - Seu Assistente de Batalha para RPG

Bem-vindo ao Lifegen! Este é um aplicativo Android desenvolvido com o objetivo de ser uma ferramenta indispensável para mestres de RPG de mesa. A ideia é simples: simplificar e agilizar o gerenciamento de combates, permitindo que o mestre se concentre no que realmente importa: a narrativa e a diversão dos jogadores.

## Status do Projeto

**Versão Atual:** 0.2.0

Este projeto está em desenvolvimento ativo. Novas funcionalidades e melhorias estão sendo adicionadas regularmente.

## O Problema que o Lifegen Resolve

Todo mestre sabe que gerenciar os pontos de vida de vários jogadores e uma horda de monstros pode ser caótico. Anotações em papel se perdem, apagar e reescrever desgasta a ficha, e outros aplicativos podem ser complexos demais. O Lifegen foi criado para ser a solução mais direta e eficiente para esse problema.

## Funcionalidades Atuais

*   **Gerenciamento de Combatentes:** Adicione, visualize e remova jogadores e NPCs da batalha de forma rápida e intuitiva.
*   **Lista de Batalha Clara:** Tenha uma visão limpa de todos os personagens envolvidos no combate.
*   **Encerramento de Batalha:** Com um único clique, limpe o campo de batalha e prepare-se para o próximo encontro, sem precisar apagar cada personagem um por um.
*   **Aplicação de Dano e Cura:** Modifique a vida dos personagens individualmente com botões de acesso rápido.

## Visão para o Futuro (Roadmap)

O Lifegen tem como objetivo se tornar um verdadeiro "canivete suíço" para o mestre de RPG. As próximas grandes funcionalidades planejadas são:

*   **Efeitos Sonoros:** Uma biblioteca de efeitos sonoros integrada para dar vida aos momentos mais épicos do seu jogo. Imagine poder tocar o som de uma espada colidindo, uma bola de fogo explodindo ou uma poção sendo bebida com um simples toque!
*   **Controle de Trilha Sonora:** Permita que o mestre crie e gerencie playlists de trilhas sonoras diretamente do aplicativo. Alterne facilmente entre uma música ambiente para exploração, uma faixa tensa para momentos de suspense e um tema épico para as batalhas.

## Construído com a Arquitetura Moderna do Android

Este projeto segue as melhores práticas de desenvolvimento Android recomendadas pelo Google, incluindo:

*   **Arquitetura MVVM (Model-View-ViewModel):** Para um código organizado, testável e de fácil manutenção.
*   **RecyclerView com ListAdapter e DiffUtil:** Para uma exibição de listas altamente eficiente e com animações automáticas.
*   **LiveData:** Para uma comunicação reativa entre a lógica de negócios e a interface do usuário.
